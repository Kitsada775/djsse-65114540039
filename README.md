**Server-Side Events (SSE)** คือเทคโนโลยีที่ช่วยให้เซิร์ฟเวอร์สามารถส่งข้อมูลแบบเรียลไทม์ไปยังไคลเอนต์ (เช่น เบราว์เซอร์) ผ่านการเชื่อมต่อ HTTP แบบเดิม โดยไม่จำเป็นต้องทำการรีเฟรชหน้าเว็บหรือส่งคำขอใหม่ทุกครั้งที่ต้องการข้อมูลใหม่

### การใช้งาน SSE เหมาะสำหรับ:
- การแสดงผลข้อมูลที่อัพเดตตลอดเวลา เช่น คะแนนการแข่งขัน, ราคาหุ้น, หรือสถานะการประมวลผล
- การส่งข้อมูลจากเซิร์ฟเวอร์ไปยังไคลเอนต์โดยไม่ต้องมีการร้องขอข้อมูลใหม่ทุกครั้ง (ใช้การเชื่อมต่อเปิดตลอด)

## วิธีการนำไปใช้ใน Django
### ขั้นตอนที่ : ติดตั้ง Django
- git clone https://github.com/your-username/djsse.git
- python -m venv venv
- venv\Scripts\activate
- pip install django
- pip install -r requirements.txt
- cd djsse
- python manage.py runserver
